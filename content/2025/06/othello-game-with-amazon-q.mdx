## Amazon Q CLIとの開発体験

### AIアシスタントの活用場面

Amazon Q CLIはbrewwを使ってインストールし、コマンドラインから直接対話できます。

```bash
$ brew install --cask amazon-q
```

早速使ってみます。

```bash
$ q
```

![amazon_q](/images/2025/06/othello-game-with-amazon-q/amazon_q.png)

Amazon Q CLIのモデルはSonetなんですね。びっくりしました。
